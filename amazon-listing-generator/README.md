# Amazon Listing Generator

## 功能概述
这是一款桌面应用程序，可根据对标Listing链接和产品图片，自动生成中英文标题、五点描述、详情描述以及处理主图和详情图。

## 主要功能
- 输入对标Listing链接和产品图片
- AI驱动的标题、五点描述、详情描述生成
- 支持中英文双语输出
- 图片处理功能（主图、详情图）
- 输出格式化为Amazon平台规范

## 技术架构
- GUI框架：Tkinter
- AI模型：集成开源大语言模型或API
- 图像处理：Pillow
- 网络请求：Requests
- 数据处理：BeautifulSoup, lxml

## 安装与使用
1. 下载exe文件
2. 运行安装
3. 输入对标链接和产品图片
4. 生成Listing内容

## 系统要求
- Windows 10/11
- 至少4GB内存
- 稳定网络连接
