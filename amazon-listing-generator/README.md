# Amazon Listing Generator

一个用于生成Amazon Listing的桌面应用程序，能够根据对标Listing链接和产品图片自动生成中英文标题、五点描述和详情描述。

## 获取程序

下载可执行文件：请参阅 [GET_EXE_FILE.md](GET_EXE_FILE.md) 了解如何获取 `AmazonListingGenerator.exe`

## 功能特性

- 输入对标Listing链接自动提取产品信息
- 支持上传多张产品图片进行处理
- 自动生成中英文标题、五点描述和详情描述
- 内置图片处理器，自动调整图片尺寸和背景
- AI驱动的内容优化建议

## 技术说明

此程序为单文件EXE，包含所有必要依赖，无需额外安装软件。
